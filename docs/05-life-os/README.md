# 05 — Life OS

**Status:** Foundation Draft

## Purpose

This folder documents the Life OS module — the set of features and
services that support personal, individual life-management functionality
on the XETROIT Platform.

## Documents That Belong Here

- Life OS feature scope and architecture
- Domain models specific to personal life-management features (e.g.
  personal goals, tracking, planning tools)
- Business rules governing this module's behavior
- Integration points between Life OS and the core platform or other
  modules

Cross-module or platform-wide concerns belong in `01-platform/` or
`02-core-platform/`, not here.
