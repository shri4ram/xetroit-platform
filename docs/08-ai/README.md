# 08 — AI

**Status:** Foundation Draft

## Purpose

This folder documents the AI Platform — how artificial intelligence
capabilities (including AI coding agents, in-product AI features, and
automation) are architected, governed, and integrated across XETROIT.

## Documents That Belong Here

- AI platform architecture (models, providers, orchestration)
- AI feature specifications for in-product functionality
- Guidelines for how AI coding agents should consume and act on platform
  documentation (see also `99-ai-context/`)
- Data, privacy, and safety considerations specific to AI features
- Business rules governing AI-driven behavior in the product

General AI-agent operating context (how agents should navigate this
documentation set) lives in `99-ai-context/`; this folder is about the AI
platform as a product/architecture capability.
