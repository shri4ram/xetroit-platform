# 10 — DevOps

**Status:** Foundation Draft

## Purpose

This folder documents infrastructure, deployment, and operational practices
for the XETROIT Platform — how code moves from commit to production and how
the platform is kept running reliably.

## Documents That Belong Here

- Infrastructure architecture (hosting, environments, networking)
- CI/CD pipeline design and standards
- Environment management (local, staging, production) and configuration
  conventions
- Monitoring, logging, and alerting strategy
- Incident response and on-call practices
- Backup, disaster recovery, and scaling strategy

Security-specific policy (as opposed to operational infrastructure)
belongs in `11-security/`.
