# 15 — Development

**Status:** Foundation Draft

## Purpose

This folder documents engineering standards and workflow practices for
developers and AI coding agents contributing code to the XETROIT Platform.

## Documents That Belong Here

- Coding standards and style guides (PHP/Laravel, Dart/Flutter, and any
  future Node.js specialized services)
- Git workflow, branching, and code review conventions
- Testing standards and requirements
- Local development environment setup
- Definition of Done and quality gates

This folder covers *how* engineering work is done; *what* to build lives
in the module-specific architecture folders and `01-platform/`.
