# 16 — Future Verticals

**Status:** Foundation Draft

## Purpose

This folder holds exploratory documentation for potential future product
lines and business verticals that are not yet committed to the platform
roadmap. It allows ideas to be captured and evaluated without implying
approval or implementation priority.

## Documents That Belong Here

- Exploratory concepts for new product verticals
- Early-stage market/opportunity notes relevant to potential future
  modules
- Speculative architecture sketches, explicitly marked as non-binding

Documents here should never reach `Approved` status while still
speculative — if a vertical becomes committed roadmap, its documentation
should graduate into a properly numbered module folder (following the
pattern of `03-business-os/` through `07-payments/`).

## Current Status

**Foundation Draft** — folder structure established, no documents authored
yet. No documents currently planned; documents will be registered in
`../DOCUMENT-INDEX.md` as they are identified.
