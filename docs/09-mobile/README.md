# 09 — Mobile

**Status:** Foundation Draft

## Purpose

This folder documents the mobile architecture for XETROIT applications,
built with **Flutter** as the platform's official cross-platform mobile
framework.

## Documents That Belong Here

- Mobile app architecture (state management, navigation, module structure)
- Flutter-specific conventions and patterns used across XETROIT mobile apps
- API/backend integration contracts consumed by mobile clients
- Mobile-specific security, offline, and performance considerations
- Release and distribution process for mobile apps

Backend API contracts consumed by mobile (but owned by the platform) belong
in `13-api/`; this folder covers the mobile client itself.
