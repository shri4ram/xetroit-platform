# 11 — Security

**Status:** Foundation Draft

## Purpose

This folder documents the security architecture and policy for the
XETROIT Platform — how the platform protects data, users, and systems
across all modules and clients.

## Documents That Belong Here

- Security architecture (authentication, authorization, encryption)
- Data protection and privacy policy
- Threat modeling and risk assessments
- Vulnerability management and disclosure process
- Compliance requirements (e.g. data residency, regulatory obligations)
- Security rules that apply platform-wide, including for AI-generated code

Payment-specific compliance belongs in `07-payments/`; this folder covers
platform-wide security concerns.
