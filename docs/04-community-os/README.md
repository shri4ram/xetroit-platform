# 04 — Community OS

**Status:** Foundation Draft

## Purpose

This folder documents the Community OS module — the set of features and
services that power community, social, and group interaction functionality
on the XETROIT Platform.

## Documents That Belong Here

- Community OS feature scope and architecture
- Domain models specific to community features (e.g. groups, memberships,
  posts, discussions, moderation)
- Business rules governing this module's behavior
- Integration points between Community OS and the core platform or other
  modules

Cross-module or platform-wide concerns belong in `01-platform/` or
`02-core-platform/`, not here.
